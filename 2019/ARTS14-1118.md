## Algorithm

### First Missing Positive

[41. First Missing Positive](https://leetcode-cn.com/problems/first-Missing-positive/)

Given an unsorted integer array, find the smallest missing positive integer.

Example 1:

    Input: [1,2,0]
    Output: 3

Example 2:

    Input: [3,4,-1,1]
    Output: 2

Example 3:

    Input: [7,8,9,11,12]
    Output: 1

Note:

Your algorithm should run in O(n) time and uses constant extra space.

```javascript
/**
 * @param {number[]} nums
 * @return {number}
 */
var firstMissingPositive = function(nums) {
    const n = nums.length;
    for (let i = 0; i < nums.length; i++) {
        if (nums[i] > n || nums[i] <= 0) {
        nums[i] = -1;
        continue;
        }
        if (nums[i] === i + 1) {
            continue;
        }
        while (nums[i] > 0 && nums[i] <= n && nums[i] !== i + 1) {
            const temp = nums[i];
            if (temp === nums[temp - 1]) {
                nums[temp - 1] === temp;
                nums[i] === -1;
                break;
            }
            nums[i] = nums[temp - 1];
            nums[temp - 1] = temp;
        }
    }
    let i;
    for (i = 0; i < n; i++) {
        if (nums[i] !== i + 1) {
            return i + 1;
        }
    }
    if (i === n) {
        return n + 1;
    }
    return 1;
};
```
